Onderaan vind je mijn recent werk. Mijn focus op lange termijn ligt in full-stack en mobiele applicaties ontwikkeling. Ik maak verschillende projecten, waaronder websites en kleine projecten en maak vaak gebruik van javascript, react of nodejs.

Mijn werk

Project 1 - Reisbureau Website (WIP) (verzonnen opdracht)

Voor het reisbureau Bon travel heb ik een website aangemaakt. Bij de front-end maak ik gebruik van TailwindCSS, HTML en Javascript. Herhalende HTML block elements zoals header of footer heb ik gesplitst in meerdere JS bestanden om ervan componenten te kunnen maken, zodat ik elke keer alleen maak gebruik van een afkorting. Op de website zijn er kleine Javascript onderdelen zoals logo slider of cookie bar. De back end wordt gemaakt met Node.js en MySQL database. Dit is nu steeds work in progress. Tot nu toe is deze website mijn grootste project. Hierbij heb ik veel nieuws geleerd (net als over reisbureaus en reizen) en het was plezier voor mij om deze website te kunnen ontwerpen. Zie het hele project op mijn Github profiel.
